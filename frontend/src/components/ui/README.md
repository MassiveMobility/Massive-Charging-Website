# UI Components

Reusable presentational primitives shared across features:

- `button.tsx` (required `styleConfig` prop)
- `input.tsx` (required `styleConfig` prop)
- `textarea.tsx` (required `styleConfig` prop)
- `select.tsx` (required `styleConfig` prop)
- `card.tsx` (required `styleConfig` prop)
- `badge.tsx` (required `styleConfig` prop)
- `heading.tsx` (required `styleConfig` prop)
- `text.tsx` (required `styleConfig` prop)

Internal route navigation uses built-in `next/link` directly with centralized link-style presets.
